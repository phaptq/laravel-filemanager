## Steps to contribute
1. Fork [phaptq/laravel-filemanager](https://github.com/phaptq/laravel-filemanager) from GitHub.
1. Run commands below:

    ```
    git clone git@github.com:phaptq/laravel-filemanager-example-5.3.git
    cd laravel-filemanager-example-5.3
    composer require phaptq/laravel-filemanager:dev-master
    make init
    ```
1. Edit codes in `vendor/phaptq/laravel-filemanager`
1. Push your changes to your fork.
1. Send a pull request to [phaptq/laravel-filemanager](https://github.com/phaptq/laravel-filemanager).
